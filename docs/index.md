# Big Data - Universidad UNIE

Bienvenido al repositorio de prácticas de la asignatura de **Big Data**.

## Objetivo del Curso
Aprender a manejar grandes volúmenes de datos optimizando el almacenamiento y el cómputo. En esta asignatura nos centraremos en formatos eficientes (Parquet), herramientas de procesamiento distribuido y buenas prácticas de ingeniería de datos.

## Práctica 1: Setup e I/O
En esta primera sesión ([ver guía](practicas/p1-setup-io.md)) nos centraremos en:

1.  **Entorno de desarrollo moderno**: Uso de `uv` para gestionar Python y dependencias de forma ultrarrápida.
2.  **Benchmark de I/O**: Compararemos CSV vs Parquet para sentir la diferencia de rendimiento en carne propia.
3.  **Filosofía**: *"Lo que no se mide, no se puede mejorar"*. Mediremos tiempo de lectura y espacio en disco.
